## 🐛 如果测试失败

### 问题 1: 仍显示"获取成员列表失败"

**解决步骤**:
1. 按 F12 打开开发者工具
2. 进入 Console 标签
3. 查看是否有错误消息
4. 检查 Network 标签中 API 请求的响应

**常见原因**:
- [ ] 后端未重启 → 重新启动后端
- [ ] 浏览器缓存 → 刷新页面 (Ctrl+F5)
- [ ] 端口被占用 → 检查 http://localhost:3001

### 问题 2: 首页仍无法显示

**检查**:
1. 前端是否正在运行? 访问 http://localhost:5173
2. 后端是否正在运行? 访问 http://localhost:3001/api/health
3. 查看浏览器控制台是否有错误

### 问题 3: 添加成员/成就后首页没有显示

**原因可能是**:
- 前端没有自动刷新 → 手动刷新页面
- API 调用有延迟 → 等待几秒后再刷新
- 数据库问题 → 检查后端日志
