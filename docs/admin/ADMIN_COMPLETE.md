## 🚀 使用方法

### 访问管理系统
```
1. 打开: http://localhost:5173
2. 点击右上角 "🔧 管理系统" 按钮
3. 输入密码: admin123
4. 登录成功！
```

### 管理团队
```
📋 战队信息 → 编辑基本信息
👥 成员管理 → 添加/编辑/删除成员
🏆 成就管理 → 记录比赛成就
📝 申请审批 → 审批入队申请
```

### 查看效果
```
编辑完成后自动保存到数据库
刷新首页 (http://localhost:5173) 即可看到最新内容
```

## 💾 数据持久化

所有数据都保存在 SQLite 数据库中：
```
backend/data.db
```

备份数据库：
```bash
cp backend/data.db backend/data.db.backup
```
