# 🚀 管理系统快速上手

## 1️⃣ 启动服务（如果还没启动）

### 终端 1 - 前端服务
```bash
cd /Users/enter/Project/yxor/frontend
npm run dev
```
访问: http://localhost:5173

### 终端 2 - 后端服务
```bash
cd /Users/enter/Project/yxor/backend
npm run dev:sqlite
```
访问: http://localhost:3001

---

## 2️⃣ 访问管理系统

### 方式 A: 通过导航栏
1. 打开首页: http://localhost:5173
2. 点击右上角"🔧 管理系统"按钮

### 方式 B: 直接访问
访问: http://localhost:5173/admin

### 登录
- 密码: `admin123`
- 点击"登录"按钮

---

## 3️⃣ 添加你的第一个成员

```
1. 登录管理系统
2. 点击"👥 成员管理"标签页
3. 点击"+ 添加新成员"按钮
4. 填写信息:
   名称: 张三
   角色: Web安全工程师
5. 点击"➕ 添加成员"
```

✅ 完成！现在返回首页就能看到新成员了

---

## 4️⃣ 添加成就

```
1. 点击"🏆 成就管理"标签页
2. 点击"+ 添加新成就"按钮
3. 填写信息:
   年份: 2024
   名称: CTF大赛
   奖项: 一等奖
4. 点击"➕ 添加成就"
```

✅ 成就已添加到首页

---

## 5️⃣ 编辑战队信息

```
1. 点击"📋 战队信息"标签页
2. 编辑以下信息:
   - 战队名称: YXOR
   - 战队描述: 我们的描述
   - Logo: (图片链接)
3. 点击"💾 保存更改"
```

---

## 6️⃣ 审批入队申请

```
1. 点击"📝 申请审批"标签页
2. 看到用户的申请信息
3. 选择操作:
   - ✅ 批准 - 通过申请
   - ❌ 拒绝 - 拒绝申请
```

---

## 💡 提示

- 📸 **头像和Logo**: 使用图片链接，如 `https://example.com/image.jpg`
- 💾 **自动保存**: 所有更改都会立即保存到数据库
- 🔄 **刷新生效**: 前端可能需要刷新才能看到最新内容
- 🔐 **安全**: 生产环境请修改默认密码！
