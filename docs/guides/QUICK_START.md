# YXOR Team - CTF战队展示网站

**前后端分离项目** | React + Express | 现代化设计

## 🎯 核心功能

✨ **战队介绍** - 大面积展示战队标志和介绍
👥 **成员展示** - 包含头像、方向、名字、签名、博客链接
🏆 **成就展示** - 展示竞赛成绩和行业认可
📧 **在线申请** - 联系邮箱和申请表单

## 🚀 快速开始

### 1. 安装依赖（推荐）
```bash
chmod +x install.sh
./install.sh
```

### 2. 或手动安装

**后端**
```bash
cd backend
npm install
npm run dev  # 运行在 http://localhost:3001
```

**前端** (新建终端)
```bash
cd frontend
npm install
npm run dev  # 运行在 http://localhost:5173
```

## 📋 项目结构
