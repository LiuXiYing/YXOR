# Logo 上传指南

## 🎨 如何上传你的战队 Logo

### 方法 1: 通过管理系统上传（推荐）

1. **打开管理系统**
   - 访问: http://localhost:5173/admin
   - 密码: admin123

2. **进入战队信息管理**
   - 点击 "📋 战队基本信息" 选项卡

3. **上传 Logo**
   - 在 "Logo 图片上传 / URL" 区域
   - 点击 "选择文件" 按钮
   - 选择你的 logo 文件（如 yxor.jpg）
   - 文件会自动转换为 Base64 格式并保存到数据库

4. **预览效果**
   - 图片上传后会显示预览
   - 点击 "💾 保存更改" 保存

5. **首页更新**
   - 刷新首页，logo 会立即显示在：
     - Hero 区域（大号 logo）
     - Navbar 上方（小号 logo）
     - TeamIntro 介绍区（中号 logo）

### 首页布局说明

**Hero 区域（首屏）**
- 只显示：Logo 和战队名字
- 简洁干净，突出你的 logo 设计

**TeamIntro 区域（向下滚动）**
- 显示详细信息：
  - ⭐ 战队标志（标语）
  - 📖 战队介绍
  - 成立年份
  - 成员数量
  - 联系方式

## 📝 logo 需求

- **格式**: JPG、PNG 或 SVG
- **大小**: 建议 500x500px 以上
- **背景**: 
  - 白色或透明背景最佳
  - 这样在不同背景上都能显示清楚

## 🔗 URL 方式上传

如果 logo 已经在线，也可以直接填入 URL：
1. 在 "Logo 图片上传 / URL" 区域的 URL 输入框
2. 粘贴你的 logo 链接（如 https://example.com/logo.png）
3. 点击 "💾 保存更改"

## 常见问题

**Q: Logo 没有显示？**
- A: 硬刷新浏览器（Cmd+Shift+R 或 Ctrl+Shift+R）
- 检查管理系统中 Logo 是否成功保存

**Q: Logo 显示很小或很大？**
- A: Logo 会自动缩放以适应容器，无需手动调整

**Q: 可以换成 URL 吗？**
- A: 可以，支持 URL 和文件上传两种方式
